# Typical Create Directory Commands

## Setup

```
mkdir jai
cd jai
```
Next lets create the sql folder
```
mkdir sql
cd sql
```

To see if you are in the right folder

```
pwd
```

Crate a SQL file

```
code . 
```

The dot represent the current directory. Execute code at the current directory

## Git


```
git init
```

This initializes git in the current folder

To check the status

```
git status
```
To add a file for source tracking

This phase is called Staging

```
git add test.sql
```
